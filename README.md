
# Finance Tracker

Finance Tracker — веб-приложение для учета личных финансов, разработанное с использованием фреймворка Bottle. Проект предназначен для управления доходами и расходами, контроля банковских счетов, постановки финансовых целей и анализа финансового состояния пользователя.

---

## Начало работы

Данные инструкции позволят получить копию проекта и запустить его на локальной машине для разработки, тестирования и использования.

Информация о развертывании представлена в разделе **«Развертывание»**.

---

## Необходимые условия

Для работы с проектом необходимо установить следующее программное обеспечение:

- Python 3.9 или выше
- pip
- Git
- MySQL Server
- Visual Studio или Visual Studio Code

Проверка версии Python:

```bash
python --version
```

Проверка версии pip:

```bash
pip --version
```

Проверка установки MySQL:

```bash
mysql --version
```

## Установка

### Шаг 1. Клонирование репозитория

```bash
git clone <ссылка_на_репозиторий>
cd FinanceTracker
```

### Шаг 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активация виртуального окружения (Windows):

```bash
venv\Scripts\activate
```

### Шаг 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 4. Запуск приложения

```bash
python app.py
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5555
```

или

```text
http://127.0.0.1:5555
```

## Развертывание

Приложение было развернуто на локальном сервере средствами встроенного WSGI-сервера библиотеки Bottle.

Запуск осуществлялся из среды разработки Visual Studio.

После запуска приложение становилось доступным по адресу:

```text
http://localhost:5555
```

что обеспечивало возможность тестирования и эксплуатации веб-интерфейса через браузер.

### Параметры развертывания

| Параметр | Значение |
|-----------|-----------|
| Сервер | Локальный WSGI-сервер Bottle |
| Порт | 5555 |
| Среда разработки | Visual Studio |
| База данных | MySQL |
| Доступ к приложению | http://localhost:5555 |

Дополнительно база данных была размещена на облачном сервере для обеспечения удаленного доступа и централизованного хранения данных.
